# Introducción a los editores {#id181OD0QE05Z}

Experience Manager Guides incluye un editor fácil de usar para crear y editar documentos estructurados. El editor oculta al escritor todas las complejidades de la estructura DITA. El editor proporciona una lista de elementos DITA que un usuario normalmente necesitaría para trabajar dentro de un documento.

## Nuevo editor

>[!NOTE]
>
> El nuevo editor está disponible en Experience Manager Guides as a Cloud Service a partir de la versión 2026.05.0. Póngase en contacto con el equipo de éxito del cliente para habilitar esta función.

El nuevo editor proporciona una evolución significativa de la experiencia de creación, con mejoras centradas en el rendimiento, el uso y la interfaz de usuario/UX. Diseñado específicamente para mapas DITA y temas DITA, permite la creación eficiente de contenido grande, complejo y altamente estructurado, a la vez que es totalmente compatible con versiones anteriores y no afecta al contenido que no sea DITA.

### Características principales

- Mejoras de rendimiento para archivos grandes: mejora la capacidad de respuesta para archivos DITA grandes con compatibilidad con marcadores de deshacer/rehacer y sucios, lo que garantiza un seguimiento de cambios fiable sin una degradación del rendimiento.
- Creación de tablas modernizadas: ofrece una experiencia de edición de tablas más intuitiva con la reordenación de arrastrar y soltar, las barras de herramientas contextuales y la inserción masiva de filas o columnas para una administración de tablas más rápida y precisa.
- Modo de autor mejorado: Proporciona visibilidad configurable de los comentarios y atributos XML directamente en el modo de autor, lo que reduce la necesidad de cambiar a la vista de Source mientras se trabaja con contenido estructurado.
- Herramientas y modos de edición avanzados: presenta el modo de autor y Source simultáneos para temas y un menú de inserción rápido, lo que permite una creación estructurada más rápida con un cambio de contexto mínimo.
- Refinamientos de la interfaz de usuario y la experiencia de usuario: mejora la legibilidad y la comodidad visual a través de un tema oscuro y CSS predeterminado más enriquecido en los modos Autor y Vista previa, lo que ofrece una interfaz de edición más moderna.
- Configuración centralizada del editor: ofrece un panel de configuración unificado para personalizar la experiencia de creación, incluido el control de etiquetas, atributos, comentarios XML, espacios de no separación y el menú de inserción rápida en el editor.

### Limitaciones del nuevo editor

En este momento, el nuevo editor no admite las siguientes funciones. Estas funcionalidades están planificadas para introducirse en próximas versiones.

- Creación con Control de cambios habilitado (cualquier cambio rastreado existente se conserva y trata como contenido normal)
- Nuevas citas (se conservan todas las citas existentes en el contenido)
- Redimensionador de imágenes en la vista Autor
- Uso de datos de fuentes de datos (por ejemplo, JIRA, bases de datos SQL, PIM o PLM)

>[!NOTE]
>
>La vista Presentación para mapas DITA no será compatible con el editor nuevo.


## Editor antiguo

El antiguo editor de Experience Manager Guides ofrece una experiencia de creación sencilla y fiable para el contenido estructurado, abstrayendo las complejidades de DITA y, al mismo tiempo, garantizando el cumplimiento de estándares admitidos como DITA 1.2, 1.3, DITA ligero y DITA especializado. Proporciona herramientas esenciales para trabajar con bloques comunes y elementos en línea, lo que lo hace adecuado para una creación de contenido coherente y controlada dentro de un marco estructurado.
