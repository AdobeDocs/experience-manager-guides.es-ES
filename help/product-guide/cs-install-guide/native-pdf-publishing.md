# Publicación nativa de PDF

Con AEM Guides, puede generar PDF de temas individuales o un archivo de mapa mediante la función de generación de resultados nativa de PDF. Puede personalizar y utilizar plantillas de PDF al generar la salida de PDF. Puede crear y diseñar diseños de página para presentar las distintas secciones de la salida de PDF según sus necesidades.

Además, AEM Guides también admite variables de idioma para ayudarle a definir cadenas localizadas. También permite definir estilos para varios elementos presentes en la salida de PDF. Si es un usuario avanzado, también puede trabajar directamente con CSS para crear diseños de página avanzados.
