# Personalizar editor {#id181AI0003PN}

AEM Guides incluye un potente editor que permite a los autores crear y editar documentos DITA. Puede personalizar la barra de herramientas del Editor para agregar o quitar cualquier función a la que tenga acceso desde la barra de herramientas. Además, puede configurar el guardado automático de archivos, la generación de ID para elementos a medida que se insertan en sus documentos y mucho más.
